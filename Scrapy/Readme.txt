Dentro deste diretório vc vê as pastas:
scrapy
wikiSpider

A pasta scrapy é o ambiente virtual com todas as dependencias do framework Scrapy. Para ativá-lo digite: source scrapy/bin/activate

A pasta wikiSpider foi criada com o comando: scrapy start project wikiSpider. Foi extraído do livro Web Scraping com Python páginas 106 a 108. Confira as páginas do livro para mais detalhes.
